ExcessiveSupport by yizzreel

Rails plugin to add a few more fancy functions to your projects

If you like ActiveSupport and you want more, you need ExcessiveSupport. It continues where ActiveSupport stopped.

Some features are:

  • Reversible questions like "not_nil?", "not_blank?".
  • Hash to hash comparisons with "include_values?".
  • Blocks as conditions statements with "and_then" and "otherwise"

Dependencies

Ruby on Rails

Install

Using git submodules: 

git submodule add git://github.com/yizzreel/excessive_support.git vendor/plugins/excessive_support
git submodule init

Using rails plugin installer: 

script/plugin install git://github.com/yizzreel/excessive_support.git

How to use

All features are documented, take a look at the RDoc.
